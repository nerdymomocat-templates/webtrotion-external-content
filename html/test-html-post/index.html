Remote HTML Demo

This file demonstrates the HTML pathway. You can author fully custom markup, then let Notion hold the metadata (title, tags, slug) while the site fetches the rendered content from GitHub.

What to test

  • Place this folder in a repo and point one Notion post at the index.html.
  • Add the repo prefix to external-content.url-prefixes.
  • Run npm run build-local and verify the post renders like any other internal page.
